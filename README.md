# 사이퍼즈 킷 시뮬레이터

**캐릭터별 최적 공격킷·방어킷을 현재 메타 통계로 계산하는 사이퍼즈 킷 시뮬레이터**

👉 **[바로 사용하기 — bk927.github.io/cyphers-damage-calculator](https://bk927.github.io/cyphers-damage-calculator/)**

![사이퍼즈 킷 시뮬레이터](public/og-image.png)

---

사이퍼즈(Cyphers)에서 **"내 캐릭터한테 지금 메타 기준 어떤 킷이 제일 합리적인지"**를 알려주는 웹 도구입니다.
캐릭터를 고르면, 사람들이 실제로 플레이하는 통계(입장률·아이템/킷 착용률·구매 순서)를 바탕으로
공격킷·방어킷별 기대 데미지와 생존력을 계산해 한눈에 비교해 줍니다.

## 주요 기능

- **공격킷 추천** — 상대 유형(딜러 / 방탱 / 회탱)별로 딜이 가장 잘 나오는 킷을 순위로
- **방어킷 추천** — 상대 필드의 공격을 얼마나 오래 버티는지(생존 사이클)로 순위
- **티어별 통계** — 전체 / 에이스 / 조커 / 다이아 / 골드 / 실버 / 브론즈, 티어마다 다른 메타 반영
- **실전 평균 상대** — 입장률로 가중한 상대 전체를 기대값으로 계산 (허수 평균 캐릭터가 아님)
- **게임 시점** — 랭커 구매 순서를 따라 초반~후반 장비 진행도별 계산
- **원사이클 처치 표시** — 궁 없이 / 궁 포함 한타 원킬 여부를 구분해 강조
- **근거 투명 공개** — 모든 숫자가 어떤 통계에서 어떻게 유도됐는지 4단계로 설명
- **모바일 지원** — 전체화면 캐릭터 선택, 반응형 레이아웃

## 데이터 출처

- **스킬 계수·기본 능력치** — 사이퍼즈 공식 홈페이지 캐릭터/스킬 페이지
- **입장률·아이템/킷 착용률** — 넥슨 공개 통계 (주간, 티어별)
- **아이템 구매 순서** — Neople 공식 API 랭커 매치 로그 집계

데이터는 GitHub Actions로 **매주 금요일(정기점검 다음날) 자동 갱신**됩니다.
계산 모델과 API에 대한 자세한 설명은 [docs/DESIGN.md](docs/DESIGN.md)를 참고하세요.

> 비공식 팬 제작 도구입니다. 방어·회피 변환은 근사치이며, 인게임 실측과 다를 수 있습니다.

## 기술 스택

Vite · React · TypeScript · GitHub Pages · GitHub Actions

## 로컬 실행

```bash
pnpm install
pnpm dev
```

데이터 재수집(선택):

```bash
node scripts/scrape-characters.mjs   # 기본 능력치
node scripts/scrape-skills.mjs        # 스킬 계수
node scripts/scrape-meta.mjs          # 티어별 착용/입장 통계
node scripts/scrape-buildorder.mjs    # 구매 순서 (NEOPLE_API_KEY 필요)
```

## 라이선스 / 고지

사이퍼즈는 넥슨/네오플의 저작물입니다. 본 프로젝트는 팬이 만든 비영리 도구로, 공개 통계와 공식 API를 활용합니다.
